# 2095. 删除链表中的中间节点

📅 4 2026.06.15

**双指针**  **Go**

---

代码和灵神是一样的，简单做个解释，为什么需要 **if** 和 **先让快指针领先**

如果快慢指针在同一起点，我觉得主要的考量有两点：

1. 当链表只有两个节点时，快慢指针各走完一步，此时 `slow` 指向最后一个节点，

   在执行删除操作时会出现 `nil = nil` 的空指针报错  

2. 我们可以发现如果两个指针同起点出发，

   其实是不能通过 `slow.Next = slow.Next.Next` 来删除的，

   因为此时 `slow` 的位置就是中间节点  

```go
func deleteMiddle(head *ListNode) *ListNode {
    if head.Next == nil {
    return nil // 只有一个节点
    }

    slow := head
    fast := head.Next.Next
    
    for fast != nil && fast.Next != nil {
    slow = slow.Next
    fast = fast.Next.Next
    }
    
    slow.Next = slow.Next.Next
    
    return head

}
```
---

对于上面的问题我们可以引入一个节点来指向 `slow`（也就是中间节点）的前一个位置  
这样快慢指针就可以同一个起点出发了，如果实在是理解不了为什么快指针要先走 `TWT`

```go
func deleteMiddle(head *ListNode) *ListNode {
    slow, fast, prev := head, head, head

    if head.Next == nil {
    return nil
    }
    
    for fast != nil && fast.Next != nil {
    prev = slow
    slow = slow.Next
    fast = fast.Next.Next
    }
    
    prev.Next = slow.Next
    return head

}
```
