# 1081. 不同字符的最小子序列

题目链接：[1081. 不同字符的最小子序列](https://leetcode.cn/problems/smallest-subsequence-of-distinct-characters/description/)

## 思路

**单调栈 + 贪心**

题目要求从 `s` 中选出一个子序列，使它包含 `s` 中所有不同字符，且每个字符只出现一次，并且字典序最小。

核心思路是用一个栈维护当前答案：

1. 从左到右遍历字符串
2. 如果当前字符已经在栈中，直接跳过，避免重复
3. 如果当前字符比栈顶字符小，并且栈顶字符后面还会出现，就弹出栈顶
4. 将当前字符加入栈中

这样做的贪心含义是：在保证每个字符都能出现一次的前提下，尽量让更小的字符出现在更靠前的位置。

## 解题过程

1. 预处理每个字符最后出现的位置 `last`
2. 遍历字符串 `s`：
   - 如果当前字符已经在栈中，跳过
   - 否则，当栈顶字符大于当前字符，且栈顶字符后面还会再次出现时，弹出栈顶
   - 将当前字符入栈，并标记为已使用
3. 返回栈中字符组成的字符串

## 代码

```cpp
class Solution {
public:
    string smallestSubsequence(string s) {
        vector<int> last(26, 0);
        for (int i = 0; i < s.size(); i++) {
            last[s[i] - 'a'] = i;
        }

        string stk;
        vector<bool> used(26, false);

        for (int i = 0; i < s.size(); i++) {
            char c = s[i];
            if (used[c - 'a']) continue;

            while (!stk.empty() && stk.back() > c && last[stk.back() - 'a'] > i) {
                used[stk.back() - 'a'] = false;
                stk.pop_back();
            }

            stk.push_back(c);
            used[c - 'a'] = true;
        }

        return stk;
    }
};
```

## 示例验证

### 示例 1

```text
s = "bcabc"

last: a=2, b=3, c=4

i=0, c='b': stk="b"
i=1, c='c': stk="bc"
i=2, c='a': 'a' < 'c' 且 last['c'] = 4 > 2，弹出 'c'
             'a' < 'b' 且 last['b'] = 3 > 2，弹出 'b'
             入栈 'a'，stk="a"
i=3, c='b': 入栈，stk="ab"
i=4, c='c': 入栈，stk="abc"
```

答案为 `"abc"`。

### 示例 2

```text
s = "cbacdcbc"

last: a=2, b=6, c=7, d=4

i=0, c='c': stk="c"
i=1, c='b': 'b' < 'c' 且 last['c'] = 7 > 1，弹出 'c'，入栈 'b'，stk="b"
i=2, c='a': 'a' < 'b' 且 last['b'] = 6 > 2，弹出 'b'，入栈 'a'，stk="a"
i=3, c='c': 入栈，stk="ac"
i=4, c='d': 入栈，stk="acd"
i=5, c='c': used['c'] = true，跳过
i=6, c='b': 'b' < 'd'，但 last['d'] = 4 <= 6，不能弹出 'd'，入栈 'b'，stk="acdb"
i=7, c='c': used['c'] = true，跳过
```

答案为 `"acdb"`。

## 复杂度分析

- 时间复杂度：`O(n)`
  - 每个字符最多入栈一次、出栈一次
- 空间复杂度：`O(1)`
  - 字符集大小固定为 26

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| `last` 数组 | 记录每个字符最后出现的位置，用来判断栈顶字符是否还能弹出 |
| `used` 数组 | 记录字符是否已经在栈中，避免重复加入 |
| 贪心弹出 | 栈顶字符大于当前字符，且栈顶字符后面还会出现时，才可以弹出 |
| 栈维护答案 | 栈中保存当前能得到的字典序最小子序列 |
