# 1260. 二维网格迁移

题目链接：[1260. 二维网格迁移](https://leetcode.cn/problems/shift-2d-grid/description/)

## 思路

**一维展开 + 索引映射**

迁移操作的本质是将二维网格展平为一维数组，然后整体右移 `k` 位，再还原为二维网格。

例如 3×3 网格：

```text
1 2 3       展平: [1,2,3,4,5,6,7,8,9]
4 5 6   →   右移1位: [9,1,2,3,4,5,6,7,8]
7 8 9       还原:
            9 1 2
            3 4 5
            6 7 8
```

核心就是把二维坐标 `(i, j)` 转成一维下标 `idx`，再计算右移后的新下标 `newIdx`，最后转回二维坐标。

## 解题过程

1. 获取网格行数 `m` 和列数 `n`，总元素数 `total = m * n`
2. 对 `k` 取模：`k %= total`，避免无效移动
3. 遍历原网格中的每个元素 `grid[i][j]`：
   - 计算一维位置：`idx = i * n + j`
   - 右移 `k` 位后的位置：`newIdx = (idx + k) % total`
   - 还原为二维坐标：`newI = newIdx / n`，`newJ = newIdx % n`
   - 赋值：`ans[newI][newJ] = grid[i][j]`
4. 返回答案 `ans`

## 代码

```cpp
class Solution {
public:
    vector<vector<int>> shiftGrid(vector<vector<int>>& grid, int k) {
        int m = grid.size();
        int n = grid[0].size();
        int total = m * n;
        k %= total;

        vector<vector<int>> ans(m, vector<int>(n));

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                int idx = i * n + j;
                int newIdx = (idx + k) % total;
                int newI = newIdx / n;
                int newJ = newIdx % n;
                ans[newI][newJ] = grid[i][j];
            }
        }

        return ans;
    }
};
```

## 示例验证

### 示例 1

```text
grid = [[1,2,3],[4,5,6],[7,8,9]], k = 1
m = 3, n = 3, total = 9, k = 1

元素移动：
  (0,0) 值 1 → idx = 0 → newIdx = 1 → (0,1)
  (0,1) 值 2 → idx = 1 → newIdx = 2 → (0,2)
  (0,2) 值 3 → idx = 2 → newIdx = 3 → (1,0)
  (1,0) 值 4 → idx = 3 → newIdx = 4 → (1,1)
  ...
  (2,2) 值 9 → idx = 8 → newIdx = 0 → (0,0)

结果：
[[9,1,2],
 [3,4,5],
 [6,7,8]]
```

### 示例 2

```text
k = 4，total = 16
k % total = 4，相当于整体右移 4 位。
```

### 示例 3

```text
grid = [[1,2,3],[4,5,6],[7,8,9]], k = 9
total = 9, k % total = 0

右移 9 位等价于不移动，结果保持原样。
```

## 复杂度分析

- 时间复杂度：`O(m × n)`
  - 每个元素遍历一次
- 空间复杂度：`O(m × n)`
  - 需要一个同等大小的答案数组

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 二维转一维 | `idx = i * n + j` |
| 右移位置 | `newIdx = (idx + k) % total` |
| 一维转二维 | `newI = newIdx / n`，`newJ = newIdx % n` |
| 取模优化 | `k %= total`，避免重复移动 |
