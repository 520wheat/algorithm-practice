# 238. 除自身以外数组的乘积

题目链接：[238. 除自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/description/)

## 思路

**前缀积 + 后缀积**

对于每个位置 `i`，答案等于：

```text
answer[i] = 左边所有元素的乘积 × 右边所有元素的乘积
```

因此可以用两次遍历完成：

1. 从左到右，先计算每个位置左边所有元素的乘积
2. 从右到左，再乘上每个位置右边所有元素的乘积

这样不需要使用除法，也可以处理数组中存在 `0` 的情况。

## 解题过程

1. 初始化答案数组 `ans`，所有位置初始为 `1`
2. 从左到右遍历：
   - `ans[i] = ans[i - 1] * nums[i - 1]`
   - 此时 `ans[i]` 表示 `nums[i]` 左边所有元素的乘积
3. 从右到左遍历：
   - 用变量 `right` 记录当前位置右边所有元素的乘积
   - 先执行 `ans[i] *= right`
   - 再更新 `right *= nums[i]`
4. 返回 `ans`

## 代码

```cpp
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> ans(n, 1);

        // 从左到右：计算前缀积
        for (int i = 1; i < n; i++) {
            ans[i] = ans[i - 1] * nums[i - 1];
        }

        // 从右到左：乘以右侧乘积
        int right = 1;
        for (int i = n - 1; i >= 0; i--) {
            ans[i] *= right;
            right *= nums[i];
        }

        return ans;
    }
};
```

## 示例验证

### 示例 1

```text
nums = [1, 2, 3, 4]

前缀积 ans：
  ans[0] = 1
  ans[1] = 1 * 1 = 1
  ans[2] = 1 * 2 = 2
  ans[3] = 2 * 3 = 6

ans = [1, 1, 2, 6]

从右到左：
  i = 3: ans[3] = 6 * 1 = 6，right = 1 * 4 = 4
  i = 2: ans[2] = 2 * 4 = 8，right = 4 * 3 = 12
  i = 1: ans[1] = 1 * 12 = 12，right = 12 * 2 = 24
  i = 0: ans[0] = 1 * 24 = 24，right = 24 * 1 = 24

结果：[24, 12, 8, 6]
```

### 示例 2

```text
nums = [-1, 1, 0, -3, 3]

前缀积 ans：
  ans[0] = 1
  ans[1] = 1 * (-1) = -1
  ans[2] = -1 * 1 = -1
  ans[3] = -1 * 0 = 0
  ans[4] = 0 * (-3) = 0

ans = [1, -1, -1, 0, 0]

从右到左：
  i = 4: ans[4] = 0 * 1 = 0，right = 1 * 3 = 3
  i = 3: ans[3] = 0 * 3 = 0，right = 3 * (-3) = -9
  i = 2: ans[2] = -1 * (-9) = 9，right = -9 * 0 = 0
  i = 1: ans[1] = -1 * 0 = 0，right = 0 * 1 = 0
  i = 0: ans[0] = 1 * 0 = 0，right = 0 * (-1) = 0

结果：[0, 0, 9, 0, 0]
```

## 图解

```text
nums = [1, 2, 3, 4]

前缀积：
  ans[0] = 1          当前位置左边没有元素
  ans[1] = 1          1
  ans[2] = 2          1 × 2
  ans[3] = 6          1 × 2 × 3

右侧乘积依次为：
  i = 3: right = 1
  i = 2: right = 4
  i = 1: right = 12
  i = 0: right = 24

最终：
  ans[3] = 6 × 1 = 6
  ans[2] = 2 × 4 = 8
  ans[1] = 1 × 12 = 12
  ans[0] = 1 × 24 = 24
```

## 复杂度分析

- 时间复杂度：`O(n)`
  - 左右各遍历一次数组
- 空间复杂度：`O(1)`
  - 不计算输出数组 `ans` 的空间

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 前缀积 | `ans[i]` 先保存 `nums[i]` 左侧所有元素的乘积 |
| 后缀积 | 用 `right` 变量累计当前位置右侧所有元素的乘积 |
| 两次遍历 | 先左后右，完成左右乘积合并 |
| 不用除法 | 可以正确处理包含 `0` 的情况 |
