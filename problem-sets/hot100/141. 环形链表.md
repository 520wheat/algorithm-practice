# 141. 环形链表

题目链接：[141. 环形链表](https://leetcode.cn/problems/linked-list-cycle/description/)

## 思路

**快慢指针**

快指针每次走 2 步，慢指针每次走 1 步。

- 如果链表有环，快指针最终会在环内追上慢指针
- 如果链表无环，快指针会先到达 `nullptr`

## 解题过程

1. 初始化 `slow = head`，`fast = head`。
2. 当 `fast != nullptr && fast->next != nullptr` 时循环：
   - `slow = slow->next`，慢指针走 1 步
   - `fast = fast->next->next`，快指针走 2 步
   - 如果 `slow == fast`，说明链表有环，返回 `true`
3. 如果循环结束，说明快指针到达链表末尾，返回 `false`。

## 代码

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    bool hasCycle(ListNode *head) {
        ListNode *slow = head;
        ListNode *fast = head;

        while (fast != nullptr && fast->next != nullptr) {
            slow = slow->next;
            fast = fast->next->next;
            if (slow == fast) {
                return true;
            }
        }

        return false;
    }
};
```

## 示例验证

### 示例 1

```text
链表：3 -> 2 -> 0 -> -4 -> 回到 2
```

执行过程：

```text
slow = 3, fast = 3
slow = 2, fast = 0
slow = 0, fast = -4
slow = -4, fast = 0
slow = 2, fast = -4
slow = 0, fast = 0，相遇
```

返回 `true`。

### 示例 2

```text
链表：1 -> 2 -> 3 -> nullptr
```

执行过程：

```text
slow = 1, fast = 1
slow = 2, fast = 3
fast->next = nullptr，退出循环
```

返回 `false`。

## 图解

```text
有环：
3 -> 2 -> 0 -> -4
     ↑         ↓
     └─────────┘

无环：
1 -> 2 -> 3 -> nullptr
```

## 复杂度分析

设链表长度为 `n`。

- 时间复杂度：`O(n)`
  - 无环时最多遍历到链表末尾；有环时快慢指针会在环内相遇
- 空间复杂度：`O(1)`
  - 只使用两个指针变量

## 关键点总结

| 操作 | 说明 |
| --- | --- |
| 快慢指针 | 快指针走 2 步，慢指针走 1 步 |
| 相遇条件 | `slow == fast` |
| 退出条件 | `fast == nullptr` 或 `fast->next == nullptr` |
| 空链表 | 会直接跳过循环并返回 `false` |
