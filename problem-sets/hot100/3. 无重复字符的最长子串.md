# 3. 无重复字符的最长子串

题目链接：[3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/)

## 思路

**滑动窗口 + 哈希表**

用两个指针 `l` 和 `r` 维护一个窗口，窗口内始终保证没有重复字符。

- `r` 不断向右扩展，将新字符加入窗口
- 如果新字符已经在窗口内出现过，移动 `l` 并移除字符，直到窗口内不再包含该字符
- 每次扩展窗口后，更新窗口长度的最大值

## 解题过程

1. 初始化 `l = 0`，`ans = 0`，用 `unordered_set` 记录窗口内的字符。
2. 遍历右指针 `r`，范围为 `0` 到 `n - 1`：
   - 如果 `s[r]` 已经在窗口中，移动 `l` 并移除 `s[l]`，直到窗口中不再包含 `s[r]`
   - 将 `s[r]` 加入窗口
   - 更新答案：`ans = max(ans, r - l + 1)`
3. 遍历结束后，返回 `ans`。

## 代码

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_set<char> window;
        int ans = 0;

        for (int r = 0, l = 0; r < s.size(); r++) {
            // 如果 s[r] 已在窗口中，移动 l 直到移除它
            while (window.count(s[r])) {
                window.erase(s[l++]);
            }

            // 将当前字符加入窗口
            window.insert(s[r]);

            // 更新答案
            ans = max(ans, r - l + 1);
        }

        return ans;
    }
};
```

## 示例验证

### 示例 1

```text
s = "abcabcbb"
```

执行过程：

```text
r = 0, s[0] = 'a': window 为空，插入 'a'，window = {a}, ans = 1
r = 1, s[1] = 'b': 无 'b'，插入 'b'，window = {a,b}, ans = 2
r = 2, s[2] = 'c': 无 'c'，插入 'c'，window = {a,b,c}, ans = 3
r = 3, s[3] = 'a': 'a' 已在窗口，移除 s[0] = 'a'，l = 1，插入 'a'，window = {b,c,a}, ans = 3
r = 4, s[4] = 'b': 'b' 已在窗口，移除 s[1] = 'b'，l = 2，插入 'b'，window = {c,a,b}, ans = 3
r = 5, s[5] = 'c': 'c' 已在窗口，移除 s[2] = 'c'，l = 3，插入 'c'，window = {a,b,c}, ans = 3
r = 6, s[6] = 'b': 'b' 已在窗口，依次移除 'a' 和 'b'，l = 5，插入 'b'，window = {c,b}, ans = 3
r = 7, s[7] = 'b': 'b' 已在窗口，依次移除 'c' 和 'b'，l = 7，插入 'b'，window = {b}, ans = 3
```

返回：

```text
3
```

### 示例 2

```text
s = "bbbbb"
```

执行过程：

```text
r = 0, s[0] = 'b': 插入 'b'，window = {b}, ans = 1
r = 1, s[1] = 'b': 'b' 已在窗口，移除 s[0] = 'b'，l = 1，重新插入 'b'，ans = 1
之后每次遇到 'b' 都会先移除旧的 'b'，再插入新的 'b'
```

返回：

```text
1
```

### 示例 3

```text
s = "pwwkew"
```

执行过程：

```text
r = 0, s[0] = 'p': window = {p}, ans = 1
r = 1, s[1] = 'w': window = {p,w}, ans = 2
r = 2, s[2] = 'w': 'w' 已在窗口，移除 s[0] = 'p'，再移除 s[1] = 'w'，插入 'w'，window = {w}, ans = 2
r = 3, s[3] = 'k': window = {w,k}, ans = 2
r = 4, s[4] = 'e': window = {w,k,e}, ans = 3
r = 5, s[5] = 'w': 'w' 已在窗口，移除旧的 'w'，插入新的 'w'，window = {k,e,w}, ans = 3
```

返回：

```text
3
```

## 复杂度分析

设字符串长度为 `n`。

- 时间复杂度：`O(n)`
  - 每个字符最多被右指针加入一次、被左指针移除一次
- 空间复杂度：`O(1)`
  - 字符集大小有限，因此哈希集合最多存储有限个字符

## 关键点总结

| 操作 | 说明 |
| --- | --- |
| 滑动窗口 | `[l, r]` 始终表示当前无重复字符的子串 |
| 哈希表 | 记录字符是否在当前窗口中 |
| 更新时机 | 每次成功扩展 `r` 后更新答案 |
