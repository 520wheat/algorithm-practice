# 70. 爬楼梯

题目链接：[70. 爬楼梯](https://leetcode.cn/problems/climbing-stairs/description/)

## 思路

**动态规划**

每次可以爬 `1` 阶或 `2` 阶。

考虑到达第 `n` 阶的最后一步：

- 如果最后一步爬 `1` 阶，那么之前在第 `n - 1` 阶
- 如果最后一步爬 `2` 阶，那么之前在第 `n - 2` 阶

所以，到达第 `n` 阶的方法数等于：

```text
到达第 n - 1 阶的方法数 + 到达第 n - 2 阶的方法数
```

也就是递推式：

```text
dp[n] = dp[n - 1] + dp[n - 2]
```

这和斐波那契数列非常类似。

由于每一项只和前两项有关，不需要保存完整的 `dp` 数组，用两个变量滚动维护即可。

## 解题过程

1. 定义 `dp[i]` 表示到达第 `i` 阶的方法数
2. 基础情况：
   - `dp[1] = 1`
   - `dp[2] = 2`
3. 从 `3` 遍历到 `n`：
   - `dp[i] = dp[i - 1] + dp[i - 2]`
4. 使用变量 `a` 表示 `dp[i - 2]`，`b` 表示 `dp[i - 1]`
5. 每次计算当前值 `c = a + b`，然后滚动更新
6. 返回 `b`

## 代码

```cpp
class Solution {
public:
    int climbStairs(int n) {
        if (n <= 2) {
            return n;
        }

        int a = 1;
        int b = 2;
        int c = 0;

        for (int i = 3; i <= n; i++) {
            c = a + b;
            a = b;
            b = c;
        }

        return b;
    }
};
```

## 示例验证

### 示例 1

```text
n = 2

方法：
1 + 1
2

返回 2
```

### 示例 2

```text
n = 3

a = 1，b = 2

i = 3：
c = a + b = 1 + 2 = 3
a = b = 2
b = c = 3

返回 3
```

对应的三种方法：

```text
1 + 1 + 1
1 + 2
2 + 1
```

## 递推过程

```text
n = 1：1
n = 2：2
n = 3：1 + 2 = 3
n = 4：2 + 3 = 5
n = 5：3 + 5 = 8
...
```

## 复杂度分析

- 时间复杂度：`O(n)`
  - 从 `3` 遍历到 `n`，每次只做常数操作
- 空间复杂度：`O(1)`
  - 只使用几个变量滚动保存状态

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 状态定义 | `dp[i]` 表示到达第 `i` 阶的方法数 |
| 最后一步 | 可以从第 `i - 1` 阶或第 `i - 2` 阶到达 |
| 转移方程 | `dp[i] = dp[i - 1] + dp[i - 2]` |
| 基础情况 | `dp[1] = 1`，`dp[2] = 2` |
| 空间优化 | 只用两个变量滚动维护前两项 |
