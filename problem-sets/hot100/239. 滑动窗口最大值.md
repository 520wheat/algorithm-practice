# 239. 滑动窗口最大值

题目链接：[239. 滑动窗口最大值](https://leetcode.cn/problems/sliding-window-maximum/description/)

## 思路

**优先队列（大根堆）+ 懒删除**

用大根堆存储窗口内的元素 `(值, 下标)`，堆顶始终是当前窗口中的最大值。

窗口滑动时，新元素入堆，然后检查堆顶：如果堆顶元素的下标不在当前窗口内，说明它已经过期，直接弹出。重复这个过程，直到堆顶元素仍然在窗口内。

## 解题过程

1. 定义大根堆 `priority_queue<pair<int, int>>`，存储 `(nums[i], i)`
2. 先将前 `k` 个元素入堆，记录第一个窗口最大值
3. 从 `i = k` 开始滑动：
   - 将 `nums[i]` 入堆
   - 检查堆顶下标是否 `< i - k + 1`，如果是则弹出
   - 堆顶即为当前窗口最大值

## 代码

```cpp
class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        priority_queue<pair<int, int>> pq;  // {值, 下标}
        vector<int> ans;

        for (int i = 0; i < k; i++) {
            pq.push({nums[i], i});
        }
        ans.push_back(pq.top().first);

        for (int i = k; i < nums.size(); i++) {
            pq.push({nums[i], i});

            while (pq.top().second <= i - k) {
                pq.pop();
            }

            ans.push_back(pq.top().first);
        }

        return ans;
    }
};
```

## 示例验证

```text
nums = [1,3,-1,-3,5,3,6,7], k = 3

初始窗口 [0,1,2]:
  pq = [(3,1), (1,0), (-1,2)]
  堆顶 (3,1) 在窗口内 → ans = [3]

i = 3，加入 (-3,3):
  pq = [(3,1), (1,0), (-1,2), (-3,3)]
  堆顶 (3,1) 在窗口 [1,3] 内 → ans = [3,3]

i = 4，加入 (5,4):
  pq = [(5,4), (3,1), (1,0), (-1,2), (-3,3)]
  堆顶 (5,4) 在窗口 [2,4] 内 → ans = [3,3,5]

i = 5，加入 (3,5):
  堆顶 (5,4) 在窗口 [3,5] 内 → ans = [3,3,5,5]

i = 6，加入 (6,6):
  堆顶 (6,6) 在窗口 [4,6] 内 → ans = [3,3,5,5,6]

i = 7，加入 (7,7):
  堆顶 (7,7) 在窗口 [5,7] 内 → ans = [3,3,5,5,6,7]
```

返回 `[3,3,5,5,6,7]`。

## 懒删除图解

```text
窗口从 [0,1,2] 滑到 [1,2,3]

堆中可能还有下标 0 的元素，且它的值很大，堆顶是它：

pq = [(100,0), (3,1), (2,2), (-1,3)]
当前窗口左边界 = 1

检查堆顶 (100,0)：下标 0 <= 0? 是 → 弹出
新的堆顶 (3,1)：下标 1 <= 0? 否 → 在窗口内 ✅
```

## 复杂度分析

- 时间复杂度：`O(n log n)`
  - 每个元素最多入堆一次，出堆一次
  - 堆中元素数量在懒删除下可能增长到 `O(n)`
- 空间复杂度：`O(n)`
  - 堆中可能保留尚未删除的过期元素

## 关键点总结

| 操作 | 说明 |
| --- | --- |
| `pq.push({nums[i], i})` | 新元素入堆 |
| `pq.top().second <= i - k` | 判断堆顶是否已过期 |
| `while` 弹出 | 懒删除所有过期堆顶 |
| 堆顶值 | 当前窗口最大值 |
