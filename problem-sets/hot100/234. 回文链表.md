# 234. 回文链表

题目链接：[234. 回文链表](https://leetcode.cn/problems/palindrome-linked-list/description/)

题解链接：[双指针](https://leetcode.cn/problems/palindrome-linked-list/solutions/3993151/shuang-zhi-zhen-by-friendly-i3enzo8f-cabr)

## 思路

**快慢指针 + 反转链表**

判断链表是否为回文，可以把链表分成前后两半，然后比较两部分是否相同。

由于链表不能像数组一样通过下标从两端向中间比较，所以可以：

1. 用快慢指针找到链表中点
2. 反转后半部分链表
3. 从头节点和后半部分头节点开始同步比较

如果每个节点值都相同，说明链表是回文链表；否则不是。

## 解题过程

1. 使用快慢指针：
   - `slow` 每次走一步
   - `fast` 每次走两步
2. 当 `fast` 走到链表末尾时，`slow` 正好到达链表中间位置。
3. 从 `slow` 开始反转后半部分链表。
4. 用两个指针分别指向：
   - `p1 = head`
   - `p2 = 反转后的后半部分头节点`
5. 同时向后遍历并比较节点值：
   - 如果存在 `p1->val != p2->val`，返回 `false`
   - 如果后半部分全部比较完成，返回 `true`

## 代码

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    bool isPalindrome(ListNode* head) {
        ListNode* slow = head;
        ListNode* fast = head;

        while (fast && fast->next) {
            slow = slow->next;
            fast = fast->next->next;
        }

        ListNode* pre = nullptr;
        ListNode* cur = slow;

        while (cur) {
            ListNode* nxt = cur->next;
            cur->next = pre;
            pre = cur;
            cur = nxt;
        }

        ListNode* p1 = head;
        ListNode* p2 = pre;

        while (p2) {
            if (p1->val != p2->val) {
                return false;
            }
            p1 = p1->next;
            p2 = p2->next;
        }

        return true;
    }
};
```

## 示例验证

### 示例 1

```text
head = [1, 2, 2, 1]
```

执行过程：

```text
快慢指针找到中点：slow 指向第二个 2

后半部分：2 -> 1
反转后：1 -> 2

前半部分从 head 开始：1 -> 2
后半部分反转后：    1 -> 2

依次比较：
1 == 1
2 == 2
```

返回 `true`。

### 示例 2

```text
head = [1, 2]
```

执行过程：

```text
快慢指针找到中点：slow 指向 2

后半部分：2
反转后：2

从 head 和反转后的后半部分开始比较：
1 != 2
```

返回 `false`。

### 示例 3

```text
head = [1, 2, 3, 2, 1]
```

执行过程：

```text
快慢指针找到中点：slow 指向 3

后半部分：3 -> 2 -> 1
反转后：1 -> 2 -> 3

前半部分从 head 开始：1 -> 2 -> 3
后半部分反转后：    1 -> 2 -> 3

依次比较都相同。
```

返回 `true`。

## 图解

```text
原链表：
1 -> 2 -> 2 -> 1

快慢指针找到后半部分起点：
1 -> 2 -> 2 -> 1
          ↑
         slow

反转后半部分：
1 -> 2    1 -> 2
↑         ↑
p1        p2

同步比较：
p1: 1 -> 2
p2: 1 -> 2
```

## 复杂度分析

设链表长度为 `n`。

- 时间复杂度：`O(n)`
  - 快慢指针遍历一次，反转后半部分一次，比较一次
- 空间复杂度：`O(1)`
  - 只使用常数个指针变量

## 关键点总结

| 操作 | 说明 |
| --- | --- |
| 快慢指针 | 找到链表中点 |
| 反转链表 | 将后半部分反转，方便从前往后比较 |
| 同步比较 | 前半部分和反转后的后半部分逐个比较 |
| 奇数长度 | 中间节点参与比较也不影响结果 |
