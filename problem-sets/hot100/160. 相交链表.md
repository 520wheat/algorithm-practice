# 160. 相交链表

题目链接：[160. 相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/description/)

题解链接：[双指针 / 长度差](https://leetcode.cn/problems/intersection-of-two-linked-lists/solutions/3985080/shuang-zhi-zhen-chang-du-chai-by-friendl-g3bx)

## 思路

**双指针 + 长度差**

如果两个链表相交，那么从相交节点开始，后面的所有节点都是同一段链表。

因此可以先分别求出两个链表的长度 `lenA` 和 `lenB`，让较长链表的指针先走 `abs(lenA - lenB)` 步。

这样两个指针到链表末尾的距离就相同了，之后同时向后移动：

- 如果两个指针指向同一个节点，说明找到了相交节点
- 如果同时走到 `nullptr`，说明两个链表不相交

注意：判断相交不能只比较节点值，而要比较节点地址，也就是 `pA == pB`。

## 解题过程

1. 遍历链表 `headA`，计算长度 `lenA`。
2. 遍历链表 `headB`，计算长度 `lenB`。
3. 令指针 `pA = headA`，`pB = headB`。
4. 如果 `lenA > lenB`，让 `pA` 先走 `lenA - lenB` 步；否则让 `pB` 先走 `lenB - lenA` 步。
5. 两个指针同时向后移动：
   - 若 `pA == pB`，返回当前节点
   - 否则继续后移
6. 如果最终走到空指针，返回 `nullptr`。

## 代码

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        int lenA = 0, lenB = 0;

        ListNode* pA = headA;
        ListNode* pB = headB;

        while (pA) {
            lenA++;
            pA = pA->next;
        }

        while (pB) {
            lenB++;
            pB = pB->next;
        }

        pA = headA;
        pB = headB;

        if (lenA > lenB) {
            int diff = lenA - lenB;
            while (diff--) {
                pA = pA->next;
            }
        } else {
            int diff = lenB - lenA;
            while (diff--) {
                pB = pB->next;
            }
        }

        while (pA && pB) {
            if (pA == pB) {
                return pA;
            }
            pA = pA->next;
            pB = pB->next;
        }

        return nullptr;
    }
};
```

## 示例验证

### 示例 1

```text
A = [4, 1, 8, 4, 5]
B = [5, 6, 1, 8, 4, 5]
相交节点值为 8
```

执行过程：

```text
lenA = 5
lenB = 6

B 比 A 长 1，所以 pB 先走 1 步。

之后 pA 和 pB 同时移动：
pA = 4, pB = 6，不同
pA = 1, pB = 1，不同
pA = 8, pB = 8，且是同一个节点，返回该节点
```

返回相交节点 `8`。

### 示例 2

```text
A = [1, 9, 1, 2, 4]
B = [3, 2, 4]
相交节点值为 2
```

执行过程：

```text
lenA = 5
lenB = 3

A 比 B 长 2，所以 pA 先走 2 步。

此时 pA 和 pB 到链表末尾的距离相同，再一起向后移动。
当 pA 和 pB 指向同一个节点 2 时，返回该节点。
```

返回相交节点 `2`。

### 示例 3

```text
A = [2, 6, 4]
B = [1, 5]
不相交
```

执行过程：

```text
lenA = 3
lenB = 2

A 比 B 长 1，所以 pA 先走 1 步。
之后两个指针同时移动，始终不会指向同一个节点。
最终一起到达 nullptr。
```

返回 `nullptr`。

## 图解

```text
链表 A：a1 -> a2 -> c1 -> c2 -> c3
链表 B：b1 -> b2 -> b3 -> c1 -> c2 -> c3

lenA = 5
lenB = 6

B 先走 1 步：
链表 A：a1 -> a2 -> c1 -> c2 -> c3
链表 B：     b2 -> b3 -> c1 -> c2 -> c3

此时两个指针到末尾的距离相同。
一起向后走，就一定会在第一个公共节点 c1 相遇。
```

## 复杂度分析

设链表 A 的长度为 `m`，链表 B 的长度为 `n`。

- 时间复杂度：`O(m + n)`
  - 需要分别遍历两个链表求长度，之后再同步遍历一次
- 空间复杂度：`O(1)`
  - 只使用了常数个指针和变量

## 关键点总结

| 操作 | 说明 |
| --- | --- |
| 求长度 | 分别计算两个链表的长度 |
| 对齐起点 | 长链表先走长度差步 |
| 同步移动 | 两个指针一起向后走 |
| 判断相交 | 比较节点地址 `pA == pB`，不是比较节点值 |
