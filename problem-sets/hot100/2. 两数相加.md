# 2. 两数相加

题目链接：[2. 两数相加](https://leetcode.cn/problems/add-two-numbers/description/)

## 思路

**模拟加法**

题目中两个链表都是按逆序存储数字的：

```text
链表头节点表示个位
后面的节点依次表示十位、百位、千位……
```

这正好符合我们平时从低位到高位做加法的顺序。

因此可以直接同时遍历两个链表，逐位相加，并用变量 `carry` 记录进位。

## 解题过程

1. 创建哑节点 `dummy`，用来简化结果链表的构造
2. 用 `cur` 指向结果链表当前末尾
3. 初始化进位 `carry = 0`
4. 当 `l1`、`l2` 或 `carry` 中任意一个存在时，继续循环：
   - 当前位初始和为 `carry`
   - 如果 `l1` 不为空，加上 `l1->val`，并让 `l1` 后移
   - 如果 `l2` 不为空，加上 `l2->val`，并让 `l2` 后移
   - 当前位的数字是 `sum % 10`
   - 新的进位是 `sum / 10`
   - 创建新节点接到结果链表后面
5. 返回 `dummy.next`

## 代码

```cpp
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode dummy;
        ListNode* cur = &dummy;
        int carry = 0;

        while (l1 != nullptr || l2 != nullptr || carry != 0) {
            int sum = carry;

            if (l1 != nullptr) {
                sum += l1->val;
                l1 = l1->next;
            }
            if (l2 != nullptr) {
                sum += l2->val;
                l2 = l2->next;
            }

            cur->next = new ListNode(sum % 10);
            cur = cur->next;
            carry = sum / 10;
        }

        return dummy.next;
    }
};
```

## 示例验证

### 示例 1

```text
l1: 2 → 4 → 3
l2: 5 → 6 → 4

表示的数字分别是：
342 和 465
```

逐位相加：

```text
carry = 0
sum = 0 + 2 + 5 = 7
当前节点 = 7，carry = 0

sum = 0 + 4 + 6 = 10
当前节点 = 0，carry = 1

sum = 1 + 3 + 4 = 8
当前节点 = 8，carry = 0
```

结果：

```text
7 → 0 → 8
```

表示数字：

```text
807
```

### 示例 2

```text
l1: 0
l2: 0

sum = 0 + 0 + 0 = 0
当前节点 = 0，carry = 0

结果：0
```

### 示例 3

```text
l1: 9 → 9 → 9 → 9 → 9 → 9 → 9
l2: 9 → 9 → 9 → 9
```

逐位相加会不断产生进位：

```text
9999999 + 9999 = 10009998
```

因为结果链表也是逆序存储，所以结果为：

```text
8 → 9 → 9 → 9 → 0 → 0 → 0 → 1
```

## 为什么循环条件要包含 `carry`？

如果最高位相加后还有进位，就需要额外创建一个节点。

例如：

```text
l1: 5
l2: 5
```

计算过程：

```text
sum = 5 + 5 = 10
当前节点 = 0
carry = 1
```

此时 `l1` 和 `l2` 都已经为空，但 `carry = 1`，还需要继续循环，创建最后一个节点 `1`。

结果是：

```text
0 → 1
```

表示数字 `10`。

## 复杂度分析

- 时间复杂度：`O(max(m, n))`
  - `m` 和 `n` 分别是两个链表的长度，需要遍历较长链表的所有节点
- 空间复杂度：`O(max(m, n))`
  - 需要创建结果链表，长度最多为 `max(m, n) + 1`

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 逆序存储 | 链表头是个位，正好从低位开始计算 |
| `carry` | 记录进位 |
| 循环条件 | `l1 != nullptr || l2 != nullptr || carry != 0` |
| 当前位 | `sum % 10` |
| 新进位 | `sum / 10` |
| 哑节点 | 简化结果链表构造 |
