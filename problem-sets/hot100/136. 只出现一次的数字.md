# 136. 只出现一次的数字

题目链接：[136. 只出现一次的数字](https://leetcode.cn/problems/single-number/description/)

## 思路

**位运算（异或）**

利用异或运算的三个性质：

- `a ^ a = 0`：相同数字异或后会抵消
- `a ^ 0 = a`：任何数字与 `0` 异或结果不变
- 异或满足交换律和结合律

因此，将数组中的所有数字依次异或，所有成对出现的数字都会抵消为 `0`，最后剩下的就是只出现一次的数字。

## 解题过程

1. 初始化 `ans = 0`。
2. 遍历数组 `nums`，依次将每个元素与 `ans` 异或。
3. 遍历结束后，返回 `ans`。

## 代码

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans = 0;
        for (int x : nums) {
            ans ^= x;
        }
        return ans;
    }
};
```

## 示例验证

```text
nums = [4, 1, 2, 1, 2]
```

异或过程：

```text
4 ^ 1 ^ 2 ^ 1 ^ 2
= 4 ^ (1 ^ 1) ^ (2 ^ 2)
= 4 ^ 0 ^ 0
= 4
```

返回：

```text
4
```

## 复杂度分析

设数组长度为 `n`。

- 时间复杂度：`O(n)`
  - 只需要遍历数组一次
- 空间复杂度：`O(1)`
  - 只使用常数个额外变量

## 关键点总结

| 表达式 | 说明 |
| --- | --- |
| `ans ^= x` | 累积异或每个元素 |
| `a ^ a = 0` | 成对出现的数字会抵消 |
| `a ^ 0 = a` | 与 `0` 异或结果不变 |
| 最终结果 | 所有重复数字抵消后，剩下只出现一次的数字 |
