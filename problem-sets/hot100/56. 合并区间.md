# 56. 合并区间

题目链接：[56. 合并区间](https://leetcode.cn/problems/merge-intervals/description/)

## 思路

**排序 + 贪心**

先按区间左端点排序，然后从前往后遍历。维护当前正在合并的区间左右边界 `[l, r]`。

如果下一个区间的左端点小于等于 `r`，说明两个区间有重叠，可以合并；否则说明当前区间已经无法继续扩展，将 `[l, r]` 加入答案，并开始维护新的区间。

## 解题过程

1. 按区间左端点排序
2. 初始化 `l` 和 `r` 为第一个区间的左右端点
3. 遍历剩余区间：
   - 如果 `intervals[i][0] <= r`，说明区间重叠，更新右边界：`r = max(r, intervals[i][1])`
   - 否则说明不重叠，将当前区间 `[l, r]` 加入答案，并更新 `l` 和 `r`
4. 遍历结束后，将最后一个区间加入答案
5. 返回答案

## 代码

```cpp
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        sort(intervals.begin(), intervals.end());
        vector<vector<int>> ans;

        int l = intervals[0][0];
        int r = intervals[0][1];

        for (int i = 1; i < intervals.size(); i++) {
            if (intervals[i][0] <= r) {
                r = max(r, intervals[i][1]);
            } else {
                ans.push_back({l, r});
                l = intervals[i][0];
                r = intervals[i][1];
            }
        }

        ans.push_back({l, r});
        return ans;
    }
};
```

## 示例验证

### 示例 1

```text
intervals = [[1,3],[2,6],[8,10],[15,18]]
排序后: [[1,3],[2,6],[8,10],[15,18]]

l = 1, r = 3

i = 1: [2,6]，2 <= 3，重叠，r = max(3, 6) = 6
i = 2: [8,10]，8 <= 6? 否，不重叠，ans = [[1,6]]，l = 8, r = 10
i = 3: [15,18]，15 <= 10? 否，不重叠，ans = [[1,6],[8,10]]，l = 15, r = 18

结束后加入最后一个区间 [15,18]
ans = [[1,6],[8,10],[15,18]]
```

## 复杂度分析

- 时间复杂度：`O(n log n)`
  - 主要开销来自排序
- 空间复杂度：`O(n)`
  - 答案数组最多存储 `n` 个区间

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 排序 | 按左端点排序，保证可能重叠的区间相邻 |
| 合并条件 | `intervals[i][0] <= r`，表示当前区间与维护区间重叠 |
| 更新右边界 | `r = max(r, intervals[i][1])` |
| 加入答案 | 不重叠时，将当前维护的 `[l, r]` 加入答案 |
