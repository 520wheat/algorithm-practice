# 20. 有效的括号

题目链接：[20. 有效的括号](https://leetcode.cn/problems/valid-parentheses/description/)

## 思路

**栈**

括号匹配具有“后出现的左括号要先匹配”的特点，也就是后进先出，因此可以用栈来处理。

遍历字符串：

- 遇到左括号，直接入栈
- 遇到右括号，检查栈顶是否是对应的左括号
  - 如果匹配，弹出栈顶
  - 如果不匹配，返回 `false`
  - 如果栈为空，也返回 `false`

遍历结束后，如果栈为空，说明所有括号都正确匹配，返回 `true`。

## 解题过程

1. 初始化一个空栈 `st`
2. 遍历字符串 `s`：
   - 遇到 `(`、`[`、`{`，入栈
   - 遇到 `)`，检查栈顶是否为 `(`，否则返回 `false`
   - 遇到 `]`，检查栈顶是否为 `[`，否则返回 `false`
   - 遇到 `}`，检查栈顶是否为 `{`，否则返回 `false`
   - 如果匹配成功，弹出栈顶
3. 遍历结束后，返回 `st.empty()`

## 代码

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char> st;

        for (char c : s) {
            if (c == '(' || c == '[' || c == '{') {
                st.push(c);
            } else {
                if (st.empty()) {
                    return false;
                }

                if (c == ')' && st.top() != '(') {
                    return false;
                }
                if (c == ']' && st.top() != '[') {
                    return false;
                }
                if (c == '}' && st.top() != '{') {
                    return false;
                }

                st.pop();
            }
        }

        return st.empty();
    }
};
```

## 更简洁的写法

也可以用哈希表存储右括号和左括号的匹配关系。

这样遇到右括号时，直接判断栈顶是否等于它对应的左括号即可。

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char> st;
        unordered_map<char, char> match = {
            {')', '('},
            {']', '['},
            {'}', '{'}
        };

        for (char c : s) {
            if (match.count(c)) {
                if (st.empty() || st.top() != match[c]) {
                    return false;
                }
                st.pop();
            } else {
                st.push(c);
            }
        }

        return st.empty();
    }
};
```

## 示例验证

### 示例 1

```text
s = "()"
```

执行过程：

```text
c = '('，入栈：['(']
c = ')'，栈顶是 '('，匹配，弹出：[]
```

结束后栈为空，返回 `true`。

### 示例 2

```text
s = "()[]{}"
```

执行过程：

```text
() 匹配
[] 匹配
{} 匹配
```

所有括号都正确匹配，返回 `true`。

### 示例 3

```text
s = "([)]"
```

执行过程：

```text
c = '('，入栈：['(']
c = '['，入栈：['(', '[']
c = ')'，栈顶是 '['，不是 '('，不匹配
```

返回 `false`。

### 示例 4

```text
s = "(]"
```

执行过程：

```text
c = '('，入栈：['(']
c = ']'，栈顶是 '('，不是 '['，不匹配
```

返回 `false`。

## 复杂度分析

- 时间复杂度：`O(n)`
  - 每个字符最多入栈一次、出栈一次
- 空间复杂度：`O(n)`
  - 最坏情况下，字符串全是左括号，栈中会存储所有字符

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 左括号 | 入栈，等待后续匹配 |
| 右括号 | 检查栈顶是否是对应左括号 |
| 栈为空 | 遇到右括号时栈为空，说明无效 |
| 遍历结束 | 栈为空才说明全部匹配 |
