# 438. 找到字符串中所有字母异位词

题目链接：[438. 找到字符串中所有字母异位词](https://leetcode.cn/problems/find-all-anagrams-in-a-string/description/)

## 思路

**固定大小滑动窗口 + 计数数组**

异位词的本质是：两个字符串包含的字母种类和数量完全相同。

因此，可以用长度为 `p.size()` 的窗口在 `s` 上滑动，维护窗口内字母的计数数组，并与 `p` 的计数数组比较。

如果两个计数数组完全相同，说明当前窗口对应的子串就是 `p` 的一个字母异位词。

## 解题过程

1. 如果 `s.size() < p.size()`，说明 `s` 中不可能存在 `p` 的异位词，直接返回空数组。
2. 统计 `p` 中每个字母的出现次数，记为 `pCnt`。
3. 初始化窗口 `[0, p.size() - 1]`，统计窗口内每个字母的出现次数，记为 `winCnt`。
4. 如果 `winCnt == pCnt`，记录起始索引 `0`。
5. 从左到右滑动窗口，每次右移一位：
   - 移除左边字符：`winCnt[s[i - m] - 'a']--`
   - 加入右边字符：`winCnt[s[i] - 'a']++`
   - 如果 `winCnt == pCnt`，记录当前窗口起始索引 `i - m + 1`
6. 返回所有记录的起始索引。

## 代码

```cpp
class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        int n = s.size(), m = p.size();
        if (n < m) return {};

        vector<int> pCnt(26, 0), winCnt(26, 0);
        vector<int> ans;

        // 统计 p 的字母频率
        for (char c : p) {
            pCnt[c - 'a']++;
        }

        // 初始化第一个窗口
        for (int i = 0; i < m; i++) {
            winCnt[s[i] - 'a']++;
        }
        if (winCnt == pCnt) {
            ans.push_back(0);
        }

        // 滑动窗口
        for (int i = m; i < n; i++) {
            winCnt[s[i - m] - 'a']--;   // 移除左边字符
            winCnt[s[i] - 'a']++;       // 加入右边字符

            if (winCnt == pCnt) {
                ans.push_back(i - m + 1);
            }
        }

        return ans;
    }
};
```

## 示例验证

### 示例 1

```text
s = "cbaebabacd", p = "abc"
```

`p` 的字母频率：

```text
pCnt: a:1, b:1, c:1
```

初始化窗口：

```text
窗口 [0,2]: "cba"
winCnt: a:1, b:1, c:1
匹配，ans = [0]
```

滑动窗口：

```text
i = 3: 移除 s[0] = 'c'，加入 s[3] = 'e'
  winCnt: a:1, b:1, e:1，不匹配

i = 4: 移除 s[1] = 'b'，加入 s[4] = 'b'
  winCnt: a:1, b:1, e:1，不匹配

i = 5: 移除 s[2] = 'a'，加入 s[5] = 'a'
  winCnt: a:1, b:1, e:1，不匹配

i = 6: 移除 s[3] = 'e'，加入 s[6] = 'c'
  winCnt: a:1, b:1, c:1，匹配，ans = [0,6]

i = 7: 移除 s[4] = 'b'，加入 s[7] = 'd'
  winCnt: a:1, c:1, d:1，不匹配
```

返回：

```text
[0, 6]
```

### 示例 2

```text
s = "abab", p = "ab"
```

`p` 的字母频率：

```text
pCnt: a:1, b:1
```

初始化窗口：

```text
窗口 [0,1]: "ab"
匹配，ans = [0]
```

滑动窗口：

```text
i = 2: 移除 s[0] = 'a'，加入 s[2] = 'a'
  winCnt: a:1, b:1，匹配，ans = [0,1]

i = 3: 移除 s[1] = 'b'，加入 s[3] = 'b'
  winCnt: a:1, b:1，匹配，ans = [0,1,2]
```

返回：

```text
[0, 1, 2]
```

## 复杂度分析

设 `n` 为字符串 `s` 的长度。

- 时间复杂度：`O(n)`
  - 每个字符在滑动窗口中被加入和移除常数次
  - 每次比较两个长度为 `26` 的数组，可视为常数时间
- 空间复杂度：`O(1)`
  - 只使用了两个长度为 `26` 的计数数组

## 关键点总结

| 操作 | 说明 |
| --- | --- |
| 固定窗口 | 窗口大小固定为 `p.size()` |
| 计数数组 | 比较窗口和 `p` 的字母频率是否一致 |
| 滑动步骤 | 每次移除左边字符，再加入右边字符 |
| `vector` 比较 | `winCnt == pCnt` 表示两个数组内容完全相同 |
