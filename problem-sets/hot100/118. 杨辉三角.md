# 118. 杨辉三角

题目链接：[118. 杨辉三角](https://leetcode.cn/problems/pascals-triangle/description/)

## 思路

**动态规划**

杨辉三角有两个基本规律：

- 每行第一个和最后一个元素都是 `1`
- 中间元素 = 上一行同位置元素 + 上一行前一个位置元素

因此可以一行一行地构造结果。

## 解题过程

1. 初始化结果数组 `ans`
2. 遍历每一行 `i`，从 `0` 到 `numRows - 1`
3. 创建长度为 `i + 1` 的数组 `row`，并全部初始化为 `1`
4. 对于中间位置 `j`，从 `1` 到 `i - 1`：
   - `row[j] = ans[i - 1][j - 1] + ans[i - 1][j]`
5. 将 `row` 加入 `ans`
6. 返回 `ans`

## 代码

```cpp
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>> ans;

        for (int i = 0; i < numRows; i++) {
            vector<int> row(i + 1, 1);

            for (int j = 1; j < i; j++) {
                row[j] = ans[i - 1][j - 1] + ans[i - 1][j];
            }

            ans.push_back(row);
        }

        return ans;
    }
};
```

## 示例验证

### 示例 1

```text
numRows = 5
```

构造过程：

```text
i = 0: [1]
i = 1: [1, 1]
i = 2: row[1] = ans[1][0] + ans[1][1] = 1 + 1 = 2 → [1, 2, 1]
i = 3: row[1] = 1 + 2 = 3，row[2] = 2 + 1 = 3 → [1, 3, 3, 1]
i = 4: row[1] = 1 + 3 = 4，row[2] = 3 + 3 = 6，row[3] = 3 + 1 = 4 → [1, 4, 6, 4, 1]
```

结果：

```text
[
  [1],
  [1, 1],
  [1, 2, 1],
  [1, 3, 3, 1],
  [1, 4, 6, 4, 1]
]
```

每个数 = 左上方 + 右上方

## 复杂度分析

- 时间复杂度：`O(numRows^2)`
  - 需要填充每一行的中间元素
- 空间复杂度：`O(numRows^2)`
  - 结果数组本身占用二维空间

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 初始化 | 每行长度为 `i + 1`，并先全部设为 `1` |
| 中间元素 | `row[j] = ans[i - 1][j - 1] + ans[i - 1][j]` |
| 边界 | 每行第一个和最后一个元素始终为 `1` |
