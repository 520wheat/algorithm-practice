# 560. 和为 K 的子数组

题目链接：[560. 和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/description/)

## 思路

**前缀和 + 哈希表**

子数组和可以用前缀和表示：

```cpp
nums[i..j] = prefix[j + 1] - prefix[i]
```

要找子数组和等于 `k`，也就是：

```cpp
prefix[j + 1] - prefix[i] = k
```

等价于：

```cpp
prefix[i] = prefix[j + 1] - k
```

所以在遍历数组时，可以用哈希表记录之前出现过的每个前缀和次数。

对于当前位置的前缀和 `pre`，只需要查找之前有多少个前缀和等于 `pre - k`，这些位置都能和当前位置组成一个和为 `k` 的子数组。

## 解题过程

1. 初始化哈希表 `mp`，令 `mp[0] = 1`
   - 表示空前缀出现过一次
   - 用来处理从数组开头开始、和正好为 `k` 的子数组
2. 遍历数组：
   - 累加当前前缀和 `pre += x`
   - 查找 `mp[pre - k]`，将出现次数累加到答案中
   - 将当前前缀和 `pre` 加入哈希表
3. 返回答案

## 代码

```cpp
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        unordered_map<int, int> mp;
        mp[0] = 1;
        int pre = 0, ans = 0;

        for (int x : nums) {
            pre += x;
            if (mp.count(pre - k)) {
                ans += mp[pre - k];
            }
            mp[pre]++;
        }

        return ans;
    }
};
```

## 示例验证

### 示例 1

```text
nums = [1, 1, 1], k = 2

pre = 0, mp[0] = 1

i = 0, x = 1: pre = 1, pre - k = -1，mp[-1] = 0，mp[1] = 1
i = 1, x = 1: pre = 2, pre - k = 0，mp[0] = 1，ans = 1，mp[2] = 1
i = 2, x = 1: pre = 3, pre - k = 1，mp[1] = 1，ans = 2，mp[3] = 1
```

返回 `2`。

### 示例 2

```text
nums = [1, 2, 3], k = 3

pre = 0, mp[0] = 1

i = 0: pre = 1, pre - k = -2，数量为 0，mp[1] = 1
i = 1: pre = 3, pre - k = 0，mp[0] = 1，ans = 1，mp[3] = 1
i = 2: pre = 6, pre - k = 3，mp[3] = 1，ans = 2，mp[6] = 1
```

返回 `2`。

## 图解

```text
nums = [1, 2, 3], k = 3

前缀和：
  pre[0] = 0
  pre[1] = 1
  pre[2] = 3
  pre[3] = 6

枚举子数组：
  [1, 2] = pre[2] - pre[0] = 3 - 0 = 3
  [3] = pre[3] - pre[2] = 6 - 3 = 3

哈希表查找 pre - k：
  pre = 3 时：找 pre - k = 0，前面有 pre[0] 一个，对应子数组 [1, 2]
  pre = 6 时：找 pre - k = 3，前面有 pre[2] 一个，对应子数组 [3]
```

## 复杂度分析

- 时间复杂度：`O(n)`
  - 只需要遍历一次数组
- 空间复杂度：`O(n)`
  - 哈希表最多存储 `n + 1` 个前缀和

## 关键点总结

| 步骤 | 说明 |
| --- | --- |
| 前缀和 | `pre` 表示当前位置的前缀和 |
| 哈希表 | 存储每个前缀和出现次数 |
| 查找条件 | 如果 `pre - k` 存在，说明有子数组和为 `k` |
| 初始化 | `mp[0] = 1` 表示空前缀，处理从开头开始的子数组 |
